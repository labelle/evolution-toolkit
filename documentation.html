How it works

Create a CSS styleguide to manage website visuals.

Evolution features a two-tiered system for managing UI elements and CSS styles.

At the center of it is the toolkit. This is where you establish your basic brand color and brand fonts.

It's also where you establish a 'UI Styleguide' of button sizes, header sizes, text sizes, etc.

Here is example code to create a header in _styleguide.scss:

.evo-header-base { // this area allows for creating base css attributes for headers.
@extend .evo-no-line; // we'll include our .evo-no-line class that we created above.
}
.evo-header-big { // this is a semantic CSS class for defining a header size.
@extend .evo-header-20; // this style lives in evolution.less file and styleguide can use any class that lives inside of evolution.less file
@extend .evo-header-base; // we'll include our .evo-header-base class that we created above.
}

The _evolution.scss file features a 'skeleton' of UI elements and neutral colors that allow you to adjust sizes and typography very quickly, simply by swapping out one element for another.

For example: evo-button-5 can quickly swap to a larger size evo-button-10.

Usage

The simplest way to start with Evolution is by including two compiled and minified CSS or just compiled CSS.

Evolution also use Bootstrap grids for responsive design.

To include bootstrap in your project you would copy and paste following code.

Bootstrap

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.6/css/bootstrap.min.css">

You will have to include an elements, styleguide, normalize and evo-icons styelsheet.

To do that, copy and paste two of the following code snippets into your project right before the closing </head> tag.

Normalize

<link rel="stylesheet" href="https://elements.humanig.com/css/evo-normalize.min.css">
<link rel="stylesheet" href="http://elements.humanig.com/css/evo-normalize.css">

Styleguide

<link rel="stylesheet" href="http://elements.humanig.com/css/styleguide.min.css">
<link rel="stylesheet" href="http://elements.humanig.com/css/styleguide.css">

Icons

<link rel="stylesheet" href="http://elements.humanig.com/css/evo-icons.min.css">
<link rel="stylesheet" href="http://elements.humanig.com/css/evo-icons.css">

Elements

<link rel="stylesheet" href="http://elements.humanig.com/css/elements.min.css">
<link rel="stylesheet" href="http://elements.humanig.com/css/elements.css">
