Usage

The simplest way to start with Evolution is by including two compiled and minified CSS or just compiled CSS.

Evolution also use Bootstrap grids for responsive design.

To include bootstrap in your project you would copy and paste following code.

Bootstrap

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.6/css/bootstrap.min.css">

You will have to include an elements, styleguide, normalize and evo-icons styelsheet.

To do that, copy and paste two of the following code snippets into your project right before the closing </head> tag.

Normalize

<link rel="stylesheet" href="https://elements.humanig.com/css/evo-normalize.min.css">
<link rel="stylesheet" href="http://elements.humanig.com/css/evo-normalize.css">

Styleguide

<link rel="stylesheet" href="http://elements.humanig.com/css/styleguide.min.css">
<link rel="stylesheet" href="http://elements.humanig.com/css/styleguide.css">

Icons

<link rel="stylesheet" href="http://elements.humanig.com/css/evo-icons.min.css">
<link rel="stylesheet" href="http://elements.humanig.com/css/evo-icons.css">

Elements

<link rel="stylesheet" href="http://elements.humanig.com/css/elements.min.css">
<link rel="stylesheet" href="http://elements.humanig.com/css/elements.css">

Icon Styles

A basic icon family designed for common website needs.

To help match your website's look and feel, these SVG icons come in three weights. Thin, light, book, and bold and 30 different sizes.

To use icons add these two classes as shown below.

<div class="evo-icon-10 evo-icon-arrow-right-book"></div>

The first class evo-icon-10 represents the size of the icon.

evo-icon- is a universal class prefix for evolution framework, and the number -10 represents size of the icon.

The second class evo-icon-arrow-right-book represents the weight of the icon.

evo-icon-arrow-right is name of the SVG file, and -book represents weight of that SVG file.

On the left side is code snippet and on the right is displayed icon of that code snippet.
